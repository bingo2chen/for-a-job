6. 如下代码输出什么

```js
var name = '123';
var obj = {
  name: '456',
  getName: function () {
      function printName () {
          console.log(this.name);
      }
      printName();
  }
}

obj.getName(); // 123
```
- 解题思路
  + 判断是不是new 绑定
    + var bar = new foo()
  + 判断是不是显性绑定 call，apply
    + var bar = foo.call(obj)
  + 判断是否在某个上下文中调用（隐式绑定）
    + var bar = obj.foo()
  + 如果都不是，就是使用默认绑定，在严格模式下绑定到undefined，否则为全局对象
    + var bar = foo()
